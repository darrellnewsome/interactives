13.8.3 Guided Practice

Mastering Python List Comprehensions

Concept The Basic Structure

[expression for item in iterable]
  • 👉 Expression: What we produce (the math or change).
  • 👉 For Item in Iterable: Where the values come from.

1️⃣ Task: Create the Source List

Create a list named numbers containing integers 1 through 10.

2️⃣ Task: Double the Numbers

Now, use a comprehension to create a new list named doubled that doubles every item in your numbers list.

Advanced 🟨 Adding a Condition

You can filter lists by adding an if statement at the end:

evens = [x for x in range(10) if x % 2 == 0]
