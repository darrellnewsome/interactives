Activity i2p13_7_3: Dictionary Unpacking

Learn how to unpack data using .items() to write cleaner, more professional code.

What is Unpacking?

person = ("Alex", 90)
name, score = person
  • The tuple contains two values.
  • We "unpack" them into two variables at once.
  • Important Rule: Order matters! The first variable gets the first value.

Unpacking with Dictionaries

Instead of manually indexing:

for key in grades:
  print(key, grades[key])

Use .items():

for name, score in grades.items():
  print(name, score)

👉 "Unpacking reduces indexing and improves readability."
