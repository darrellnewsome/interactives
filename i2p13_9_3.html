Combining Data Structures

Python Practice: Dictionaries and Lists

🟦 What Is Happening?

  • • A Dictionary stores student names (Keys)
  • • Each student is connected to a List of scores (Values)
  • • Programs combine structures to organize complex data.

🟨 Accessing the Data

print(class_grades["Alex"])

Returns: [90, 88, 92]

"Alex" is the key; the value is the whole list.

🟢 Looping with .items()

for student, scores in class_grades.items():
    print(student, scores)

The .items() method allows us to grab both the key (student) and the value (the list of scores) at the same time.
